# xmbuffer
动态扩展buffer的读写库,主要用于消息包的编解码处理

# 主要模块说明
- XBuffer 动态扩展的buffer
- XPackageReader 用于读取buffer在读取器
- XPackageWriter 用于与入buffer的定入器
